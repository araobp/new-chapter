# New Chapter

社会人になって36年経った。

次のステップへ進むため、36年間で獲得した私自身の専門技術スキルを特定したい。

## 技術的な経歴

- 電子交換機の共通線信号方式部分開発：リアルタイム信号処理が専門な組み込み開発（MCUのファームウエア開発に相当）
- モバイルエージェント: 分散コンピューティング、P2P
- SIP: 分散コンピューティング、P2P、マルチメディア信号処理、ユーザインタフェース、センサー処理(CODEC)
- SIPが関わる各種通信サービスの考案、Unified Communications、テレプレゼンス
- SDN: 分散コンピューティング、分散データベース（NOSQL)、分散エージェント
- IoT: センサー処理（９軸センサー、赤外線アレイセンサー、マイクセンサー、カメラセンサー）とデータ可視化
- AI: 環境音、物体検出（CNN)
- Android: ユーザインタフェース、タッチパネル、各種センサー
- Unity/Blender: 3DCG, 測量, VR, AR, ユーザインタフェース、各種センサー
- マーテック： UX（体験型顧客体験）、６面240度スクリーン向け裸眼VR動画の企画開発、ユーザインタフェース、データ分析・データ検索と可視化
- AIエージェント：Gemini APIで各種AIアプリ開発、現在はClaude Coworkのスキル開発

## 総合的に捉えると

- 各種センサー、3DCGやAIといった部品技術のスキル
- 分散処理やP2P
- 物理への興味
